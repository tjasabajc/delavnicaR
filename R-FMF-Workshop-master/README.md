# R-FMF-Workshop
Materials for the R workshop at Faculty of Mathematics and Physics.

Requirements:

- R (https://www.r-project.org/)
- RStudio (https://www.rstudio.com/)
- R packages:
  - ggplot2
  - tidyr
  
To install a R package, use `install.packages("packageName")` in the R or
RStudio console.
